-- SETTINGS -- (Buka RCNN_TESTING buat lebih ngerti)
DETECTION THRESHOLD -> Seberapa pastinya AI merasa pada prediksinya
IOU THRESHOLD -> Kalkulasi antara kotak yang dibuat oleh AI dan Kotak yang dibuat user
CONFIRMATION TIME -> Waktu delay ketika suatu object dideteksi pada lot

*iou dan confirmation hanya ada pada parking detection (lot)
*akun sebelah settings bisa dipencet

-- STEP-STEP JALAN PROGRAM --
1. Menyalakan database dengan XAMPP, tekan tombol start pada Apache dan MySQL di XAMPP.
2. Menghidupkan program dengan membuka terminal (Ctrl + `), memasukkan "uvicorn main:app --reload --host 0.0.0.0 --port 8000" dan menekan enter.
3. Ke Chrome dan ke halaman http://127.0.0.1:8000/
4. Login menggunakan username "admin" dan password "admin" atau register user baru.
5. Untuk sekarang, sudah terdaftar 2 kamera (Kamera Lantai 4) yang sama, dengan setting lot (Parking) dan counting (Tripwire).
6. Saat menyalakan program, kamera dan detektor langsung nyala jadi bisa langsung tekan "Launch Stream" untuk ke dashboard.
7. Jika ingin daftar kamera baru harus dalam format "rtsp://username:password@ip_address:port/stream_path", contoh kamera lantai 4: "rtsp://admin:admin@192.168.0.187:554/11". (Bisa gunakan ODM untuk cek stream path)
8. Untuk menggambar kotak, harus membuat zona dan klik zona terlebih dahulu.
9. Kotak akan terbentuk dari 4 titik. Klik untuk membuat titik.
10. Ada slider diatas untuk ganti setting kamera lot (Parking) dan counting (Tripwire).
11. Ketika di dashboard, tombol "Edit Bounds" untuk balik ke halaman untuk edit kotak.
12. Jika setting kamera counting (Tripwire), maka ada tombol "Set Spaces" untuk mengedit jumlah parkiran per zona (Kalau bisa jangan langsung edit jumlah dari database karena harus reload program).
13. Terdapat settings di kanan atas untuk mengatur deteksi AI.
14. Sebelah setting ada tombol akun dengan initals dari username (admin -> A), untuk logout atau balik ke halaman untuk daftar kamera baru.
15. Setting pada kamera lot (Parking) ada 3, "Detection Threshold", "IOU Threshold", dan "Confirmation Time". Sedangkan pada kamera counting (Tripwire) hanya ada "Detection Threshold".
